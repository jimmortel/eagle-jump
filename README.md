# 🦅 Eagle-Jump

**Eagle-Jump** est un jeu d'ascension verticale infini, inspiré des classiques du genre, mais entièrement repensé pour l'écosystème **eagle-pi**.

## 🚀 Concept
L'objectif est simple : monter le plus haut possible en sautant sur des plateformes en forme d'ailes d'aigle. Plus vous montez, plus l'atmosphère change et plus le défi devient intense.

## ✨ Caractéristiques
- **Progression Dynamique** : Changement de couleurs de fond selon l'altitude (Aube -> Zénith -> Crépuscule -> Espace).
- **Économie Web3** : Intégration de paiements en **$TYSM**, **SOL** et **USDC**.
- **Boutique de Skins** : Personnalisez votre aigle au "Nid".
- **Easter-Egg** : Saurez-vous trouver "L'Oeil du Cyclone" ?

## 🛠️ Stack Technique
- **Langage** : HTML5 / JavaScript (Canvas API)
- **Optimisation** : Conçu spécifiquement pour mobile (**Oppo Find X3 Pro**) et les Frames Farcaster.
